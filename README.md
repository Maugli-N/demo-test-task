```
Демонстрация выполнения тестового задания.
Текст задания в документе Task.pdf

Демонстрация выполнятеся на стенде
    FreeBSD 9.0-RELEASE
    PERL v.5.8.9
    MySQL v5.1.60
    Apache v1.3.41

Список файлов проекта
    Task.pdf           - тестовое задание
    maillog            - тестовые данные
    init-tables.sql    - конфигурация базы данных, адаптированная
                         под MyISAM таблицы MySQL версии стенда
    to_db.pm           - общая библиотека для соединения с базой
                         данных
    mail-log-parser.pl - парсер журнала почтовых сообщений
    show-records.cgi   - обработчик форма поиска записей
    show-records.data  - HTML часть этой формы


По этому заданию созданы

 1. Парсер журнала почтовых сообщений

    программа - mail-log-parser.pl
    язык программирования - perl
    файл тестовых данных - maillog
    конфигурация базы данных - init-tables.sql
    инициализация базы данных

        cat init-tables.sql|mysql demo_to_gu_ru

    параметры запуска парсера из консоли

        ./mail-log-parser.pl maillog


 2. Форма поиска записей журнала почтовых сообщений
    по части названия почтового адреса

    программа - show-records.cgi
    язык программирования - perl
    тестовые данные - таблицы message и log,
                      заполненные парсером
    ссылка на форму

        http://demo.to-go.ru/show-records.cgi

    В дополнение к выданному заданию на форму добавлена
    опция поиска записей в таблице «log», чтобы показать,
    что по внутренним id сообщений, отобранным по почте
    из таблице «log», в таблице «message» находятся записи
    от обработки сообщений с найденными внутренними id, но
    не всегда от тех, которыые содержали именно искомый
    адрес электронной почты.
```
